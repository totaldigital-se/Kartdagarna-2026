Workshop at Kartdagarna 2026: Beyond the Syntax Wars: Uniting Python & R for risk management

Tired of the Python vs. R trench war? In this fast-paced, hands-on workshop we call a truce and then cash in on it. You’ll learn how Python and R amplify each other to crush real problems in risk management using remote sensing, machine learning and stats. We’ll build end-to-end, reusable workflows, switch languages without friction, and show where each one shines.  
Come for the peace treaty, stay for the power boost. Bring a laptop, your curiosity, and leave with results, not rivalries.
